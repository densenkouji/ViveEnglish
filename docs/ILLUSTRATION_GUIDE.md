# 挿絵プロンプト運用ガイド

ViveEnglish の各レッスンには「挿絵」タブがあり、画像生成AI用のプロンプトを表示します。
ここでは、その仕組みと、好みのテイストに揃えるための運用方法をまとめます。

## 既定の挿絵（差し替え用プレースホルダー）

全12レッスンに、フラットなベクター調のプレースホルダー挿絵（SVG）を同梱しています。
ファイルは次の場所にあり、レッスンIDと同名です。

```
web/illustrations/<lesson_id>.svg
   例) web/illustrations/school-01.svg, travel-01.svg, culture-02.svg ...
```

アプリはレッスン一覧カードと「挿絵」タブでこのSVGを表示します。

**差し替え方法：** 同じファイル名で好きな画像に置き換えるだけです（再起動不要、ブラウザ再読込でOK）。
- SVGのまま編集してもよいですし、PNG/JPGに変えたい場合は同名の別形式に置き換え、
  必要なら拡張子の対応を `app/content/lessons.json` の各レッスン `illustration.image` で指定できます
  （未指定時は `/illustrations/<lesson_id>.svg` を使用）。
- 画像が見つからない場合、カードはテーマ絵文字に自動フォールバックします。

下記の「画像生成プロンプト」で生成した画像に差し替える、という使い方を想定しています。

## 仕組み

挿絵プロンプトは2つの要素を合成して作られます。

```
最終プロンプト = レッスン固有のシーン記述 ＋ 選択中のアートスタイルの suffix
```

- **シーン記述（scene）**：レッスンごとに `lessons.json` の `illustration.scene` に英語で定義。
  「誰が・どこで・何をしているか」を1文で表します。
- **アートスタイル（suffix）**：`art_styles.json` のプリセット。利用者が**全レッスン共通**で選びます。

これにより「テイストは利用者ごとに好みが分かれる」要件に対応し、設定画面の1か所を変えるだけで
全レッスンの挿絵の雰囲気が一括で切り替わります。ネガティブプロンプトもスタイルごとに付与されます。

## 用意しているスタイル（プリセット）

| キー | 名称 | 想定 |
|---|---|---|
| `watercolor_picturebook` | やわらかい水彩・絵本風（**既定**） | 全年齢に親しみやすい温かいトーン |
| `flat_modern` | フラット・モダンなベクター | 清潔感・ビジネス寄り |
| `anime_manga` | アニメ・マンガ風 | 若年層向け |
| `warm_pencil` | やさしい色鉛筆スケッチ | 素朴・落ち着いた雰囲気 |

切り替え：アプリの「設定」→「挿絵スタイル」。即座に全レッスンへ反映されます。

## 使い方（手順）

1. レッスンの「挿絵」タブを開く。
2. 表示された英語プロンプトを「📋 コピー」。
3. 画像生成AI（DALL·E / Stable Diffusion / Foundry Local の画像モデル等）に貼り付け。
4. ネガティブプロンプト欄も対応モデルに設定。
5. 推奨アスペクト比（既定 3:2）で生成。

## スタイルを追加・調整する

`app/content/art_styles.json` の `presets` にエントリを追加します。

```json
"my_style": {
  "name_ja": "表示名",
  "description_ja": "設定画面に出る説明",
  "prompt_suffix": "画風を表す英語の語句（カンマ区切り）, no text, all-ages friendly",
  "negative_prompt": "避けたい要素"
}
```

共通で必ず入れている指定：`no text or letters`（画像内に文字を入れない）、`all-ages friendly`、`single clear focal subject`。
学習用途のため、文字なし・全年齢向け・主役が明確、を全スタイルで担保しています。

## シーン記述を書くときのコツ

- 1文・英語・現在進行形で「人物＋場所＋動作＋雰囲気」。
- 固有名詞や文字（看板の文章など）は避ける。
- レッスンのテーマが一目で伝わる小物を1〜2点だけ添える（例：空港なら「suitcase」「departure board」）。
